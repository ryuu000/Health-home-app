A React frontend with Python Flask backend provides a scalable alternative to OutSystems for a MAX@Home sample app, enabling patient management, nurse scheduling, and home care services via REST APIs that integrate with external services like payments or EHR systems.[1][2][3]

## Tech Stack
React handles responsive UIs for dashboards and mobile views, while Flask manages CRUD operations, authentication, and real-time scheduling with SQLAlchemy for data persistence.[4][1] Use JWT for secure role-based access (patient, clinician, admin) and SocketIO for live updates on visits or vitals.[2][5] Integrate third-party APIs like Razorpay for payments, Google Maps for Delhi NCR geolocation, and Twilio for SMS notifications.[3][2]

## Core Features
- Patient registration with medical history, consent uploads, and vitals tracking.
- Booking calendar for ICU-at-home, physiotherapy, and nursing with slot availability.
- Clinician portal for task assignment, check-ins, and compliance checklists.
- Admin dashboard with utilization charts and reports.[1][2]


```python
# backend/app.py
from flask import Flask, request, jsonify
from flask_sqlalchemy import SQLAlchemy
from flask_jwt_extended import JWTManager, jwt_required, create_access_token
from flask_cors import CORS
import os
from datetime import timedelta

app = Flask(__name__)
app.config['SQLALCHEMY_DATABASE_URI'] = 'mysql://user:pass@localhost/maxathome'
app.config['JWT_SECRET_KEY'] = os.getenv('JWT_SECRET')
db = SQLAlchemy(app)
jwt = JWTManager(app)
CORS(app)

# Models (simplified)
class Patient(db.Model):
    id = db.Column(db.Integer, primary_key=True)
    name = db.Column(db.String(100))
    phone = db.Column(db.String(15))
    address = db.Column(db.Text)  # Delhi NCR focus
    medical_history = db.Column(db.Text)

class Booking(db.Model):
    id = db.Column(db.Integer, primary_key=True)
    patient_id = db.Column(db.Integer, db.ForeignKey('patient.id'))
    service = db.Column(db.String(50))  # 'ICU-at-home', 'physiotherapy'
    datetime = db.Column(db.DateTime)
    clinician_id = db.Column(db.Integer)

# Routes
@app.route('/register', methods=['POST'])
def register():
    data = request.json
    patient = Patient(name=data['name'], phone=data['phone'], address=data['address'])
    db.session.add(patient)
    db.session.commit()
    return jsonify({'id': patient.id}), 201

@app.route('/login', methods=['POST'])
def login():
    data = request.json
    # Validate user (simplified)
    access_token = create_access_token(identity=data['phone'], expires_delta=timedelta(hours=24))
    return jsonify(access_token=access_token)

@app.route('/bookings', methods=['POST', 'GET'])
@jwt_required()
def bookings():
    if request.method == 'POST':
        data = request.json
        booking = Booking(patient_id=data['patient_id'], service=data['service'], datetime=data['datetime'])
        db.session.add(booking)
        db.session.commit()
        return jsonify({'id': booking.id}), 201
    # GET all bookings with geolocation logic for Delhi NCR
    bookings_list = Booking.query.all()
    return jsonify([{'id': b.id, 'service': b.service} for b in bookings_list])

# External API integration example (Razorpay payments)
@app.route('/pay', methods=['POST'])
@jwt_required()
def create_payment():
    # Integrate Razorpay API here
    return jsonify({'payment_id': 'razorpay_xxx'})

if __name__ == '__main__':
    with app.app_context():
        db.create_all()
    app.run(debug=True, port=5000)
```
Run with `python app.py` after setting up MySQL database.[1][2]

## Frontend Setup (React)
Create app and key components.

```bash
npx create-react-app frontend
cd frontend
npm i axios react-router-dom recharts date-fns
npm start
```

```jsx
// frontend/src/App.js
import React from 'react';
import { BrowserRouter as Router, Routes, Route } from 'react-router-dom';
import Dashboard from './components/Dashboard';
import PatientProfile from './components/PatientProfile';
import Booking from './components/Booking';
import { AuthProvider } from './context/AuthContext';

function App() {
  return (
    <AuthProvider>
      <Router>
        <div className="App">
          <Routes>
            <Route path="/" element={<Dashboard />} />
            <Route path="/patient/:id" element={<PatientProfile />} />
            <Route path="/book" element={<Booking />} />
          </Routes>
        </div>
      </Router>
    </AuthProvider>
  );
}
export default App;
```

```jsx
// frontend/src/components/Booking.js (example)
import React, { useState } from 'react';
import axios from 'axios';
import { useAuth } from '../context/AuthContext';

const Booking = () => {
  const [service, setService] = useState('');
  const [date, setDate] = useState('');
  const { token } = useAuth();

  const handleBook = async () => {
    try {
      await axios.post('http://localhost:5000/bookings', 
        { service, datetime: date, patient_id: 1 },
        { headers: { Authorization: `Bearer ${token}` } }
      );
      alert('Booking confirmed!');
    } catch (error) {
      console.error(error);
    }
  };

  return (
    <div>
      <h2>Book Service (Delhi NCR)</h2>
      <select value={service} onChange={(e) => setService(e.target.value)}>
        <option>ICU-at-home</option>
        <option>Physiotherapy</option>
        <option>Nursing</option>
      </select>
      <input type="datetime-local" value={date} onChange={(e) => setDate(e.target.value)} />
      <button onClick={handleBook}>Book Now</button>
    </div>
  );
};
export default Booking;
```
